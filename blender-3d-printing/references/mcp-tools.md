# MCP Tools Reference

Complete reference for all available Blender MCP tools.

## Scene and Object Information

### mcp__blender__get_scene_info
Get overview of current scene including objects, collections, and general scene properties.

### mcp__blender__get_object_info
Get detailed information about a specific object by name.

**Parameters:**
- `object_name`: Name of the object to inspect

### mcp__blender__get_viewport_screenshot
Capture the current 3D viewport view as an image.

**Parameters:**
- `max_size`: Maximum size in pixels for the largest dimension (default: 800)

## Code Execution

### mcp__blender__execute_blender_code
Execute arbitrary Python code in Blender.

**Parameters:**
- `code`: Python code to execute

**IMPORTANT**: Break complex operations into smaller steps. Execute code incrementally to avoid errors.

## Polyhaven Integration

### mcp__blender__get_polyhaven_status
Check if Polyhaven integration is enabled in Blender.

### mcp__blender__get_polyhaven_categories
Get available asset categories for a specific asset type.

**Parameters:**
- `asset_type`: Type of asset (hdris, textures, models, all)

### mcp__blender__search_polyhaven_assets
Search for assets on Polyhaven with optional filtering.

**Parameters:**
- `asset_type`: Type of assets to search for (hdris, textures, models, all)
- `categories`: Optional comma-separated list of categories to filter by

### mcp__blender__download_polyhaven_asset
Download and import a Polyhaven asset into Blender.

**Parameters:**
- `asset_id`: ID of the asset to download
- `asset_type`: Type of asset (hdris, textures, models)
- `resolution`: Resolution to download (e.g., 1k, 2k, 4k)
- `file_format`: Optional file format (e.g., hdr, exr for HDRIs; jpg, png for textures; gltf, fbx for models)

### mcp__blender__set_texture
Apply a previously downloaded Polyhaven texture to an object.

**Parameters:**
- `object_name`: Name of the object to apply the texture to
- `texture_id`: ID of the Polyhaven texture to apply (must be downloaded first)

## Sketchfab Integration

### mcp__blender__get_sketchfab_status
Check if Sketchfab integration is enabled in Blender.

### mcp__blender__search_sketchfab_models
Search for models on Sketchfab with optional filtering.

**Parameters:**
- `query`: Text to search for
- `categories`: Optional comma-separated list of categories
- `count`: Maximum number of results to return (default 20)
- `downloadable`: Whether to include only downloadable models (default True)

### mcp__blender__download_sketchfab_model
Download and import a Sketchfab model by its UID.

**Parameters:**
- `uid`: Unique identifier of the Sketchfab model

**Note**: Model must be downloadable and you must have proper access rights.

## Hyper3D Rodin (AI Generation)

### mcp__blender__get_hyper3d_status
Check if Hyper3D Rodin integration is enabled and get the key type (MAIN_SITE or FAL_AI).

**Important**: Remember the key type for later use in polling and importing.

### mcp__blender__generate_hyper3d_model_via_text
Generate a 3D model from a text description.

**Parameters:**
- `text_prompt`: Short description of the desired model in English
- `bbox_condition`: Optional list of 3 floats controlling proportions [Length, Width, Height]

**Returns**: Job ID for polling status

### mcp__blender__generate_hyper3d_model_via_images
Generate a 3D model from image references.

**Parameters:**
- `input_image_paths`: Absolute paths of input images (list) - Required for MAIN_SITE mode
- `input_image_urls`: URLs of input images (list) - Required for FAL_AI mode
- `bbox_condition`: Optional list of 3 floats controlling proportions [Length, Width, Height]

**Note**: Only provide one of {input_image_paths, input_image_urls} based on the key type.

**Returns**: Job ID for polling status

### mcp__blender__poll_rodin_job_status
Check if the Hyper3D Rodin generation task is completed.

**For MAIN_SITE mode:**
- **Parameters**: `subscription_key` from the generate step
- **Returns**: List of status values ("Done", "Failed", "Canceled")

**For FAL_AI mode:**
- **Parameters**: `request_id` from the generate step
- **Returns**: Status ("COMPLETED", "IN_PROGRESS", "IN_QUEUE", or failed state)

**Note**: This is a polling API - only proceed when status is final.

### mcp__blender__import_generated_asset
Import the asset generated by Hyper3D Rodin after generation is complete.

**Parameters:**
- `name`: Name for the object in the scene
- `task_uuid`: For MAIN_SITE mode - from generate step
- `request_id`: For FAL_AI mode - from generate step

**Note**: Only provide one of {task_uuid, request_id} based on the key type.

## Hunyuan3D (AI Generation)

### mcp__blender__get_hunyuan3d_status
Check if Hunyuan3D integration is enabled and get the key type.

**Important**: Remember the key type for later use.

### mcp__blender__generate_hunyuan3d_model
Generate a 3D model using Hunyuan3D from text and/or image input.

**Parameters:**
- `text_prompt`: Optional short description in English or Chinese
- `input_image_url`: Optional local or remote image URL

**Note**: Can provide both text and image for better results.

**Returns**: Job ID (format: "job_xxx")

### mcp__blender__poll_hunyuan_job_status
Check if the Hunyuan3D generation task is completed.

**Parameters:**
- `job_id`: Job ID from the generate step

**Returns:**
- Status: "DONE", "RUN", or failed state
- When "DONE": Includes `ResultFile3Ds` with ZIP file path

**Note**: This is a polling API - only proceed when status is final.

### mcp__blender__import_generated_asset_hunyuan
Import the asset generated by Hunyuan3D after generation is complete.

**Parameters:**
- `name`: Name for the object in the scene
- `zip_file_url`: ZIP file URL from the poll result
